1. Sklonuj repozytorium.
2. Zainstaluj wymagane biblioteki: `pip install -r requirements.txt`
3. Wykonaj migracje bazy danych: `python manage.py migrate`
4. Uruchom serwer: `python manage.py runserver`
5. Aplikacja dostępna pod adresem: `http://127.0.0.1:8000/`
